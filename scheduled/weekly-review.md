# Weekly Review

**Schedule:** Friday at 5:00 PM
**Duration:** ~10-15 minutes
**Purpose:** Comprehensive weekly portfolio review for the founder

## Instructions

You are generating the weekly board review.

1. Execute the `weekly-review` skill.

2. This will:
   - Gather all daily digests from this week
   - Read all venture states in depth
   - Compare to last week's review for trends
   - Generate a comprehensive board-level review
   - Save to `digests/weekly/{year}-W{week}.md`
   - Update Notion Digests database
   - Extract any new cross-venture learnings to `knowledge/lessons.md`

3. The weekly review is the founder's primary strategic document. It should:
   - Provide portfolio-level metrics with week-over-week trends
   - Deep dive into each venture
   - Surface cross-venture patterns and insights
   - Make strategic recommendations
   - List all pending founder decisions
   - Look ahead to next week's priorities

4. **Ecosystem Updates** — Include a section covering:
   - New discoveries this week (from the latest scout sweep)
   - Top recommendations for active ventures
   - Any tools that were deprecated or went stale
   - Emerging tools worth watching (Score 50-70)
   - Read `knowledge/ecosystem/_index.md` and `_recommendations.md` for this data
