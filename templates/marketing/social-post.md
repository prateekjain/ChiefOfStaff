# Social Media Post Template

Use this template when drafting social media posts for approval.

## Format

```markdown
**Platform:** {{PLATFORM}} (Twitter/X, LinkedIn, Reddit, etc.)
**Venture:** {{VENTURE_SLUG}}
**Character count:** {{COUNT}} / {{LIMIT}}

---

{{POST_CONTENT}}

---

**Hashtags:** {{HASHTAGS}} (if applicable)
**Link:** {{URL}} (if applicable)
**Image:** {{IMAGE_DESCRIPTION}} (if applicable)
**Best time to post:** {{TIMING}} (based on audience research)
```

## Guidelines

- **Twitter/X:** Max 280 characters. Lead with a hook. Use 1-3 hashtags max.
- **LinkedIn:** Can be longer (up to 3,000 chars). Professional tone. Tag relevant people/companies.
- **Reddit:** Must provide genuine value to the community. No overt self-promotion. Follow subreddit rules.
- **All platforms:** Factual claims only. No exaggeration. Transparent about AI if relevant.

## Submission

After drafting, submit to the Decisions DB via the `submit-for-approval` skill:
- Type: Content
- Include the full post text in Context
- Note the target platform and timing
